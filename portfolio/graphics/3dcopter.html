Controls:

  • Control the third aircraft with keyboard
  • Left arrow moves +x direction
  • Right arrow moves -x direction
  • Down arrow moves +z direction
  • Up arrow moved -z direction
  • + key will increase speed the third aircraft flips (hold down)
  • - key will decrease speed the third aircraft flips (hold down)
  • R key resets position, speed, and color
  • C key will give it and it's dish a new random color
  • I did not check bounds so you can move it off screen potentially
